# Quiz App

A web-based quiz application with gamification features.
https://quiz-time-kevin.netlify.app/

## Setup
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the development server.

## Features
- Start Quiz
- Results summary
